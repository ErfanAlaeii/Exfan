سازنده اکسل خالق: عرفان — راهنمای نصب و استفاده
=====================================

نصب
----
1. فایل ExcelCreator-Setup-1.0.0-win-x64.exe را اجرا کنید.
2. مراحل نصب را تأیید کنید (نیازی به نصب جداگانه .NET نیست).
3. از منوی شروع «سازنده اکسل خالق: عرفان» را باز کنید.

استفاده
-------
1. نوع تقویم (میلادی یا شمسی) را در صفحه اصلی انتخاب کنید.
2. یک الگو (مثلاً پیگیری وظایف) را انتخاب کنید.
3. نام فایل را وارد کنید و مراحل ویزارد را طی کنید.
4. فایل .xlsx را ذخیره کنید — می‌توانید با Microsoft Excel باز کنید.

پشتیبانی
--------
- تنظیمات در: %AppData%\ExcelCreator\settings.json
- الگوهای قابل ویرایش پس از نصب در پوشه نصب: Templates\

نیازمندی‌ها
-----------
- Windows 10 یا 11 (64-bit)
- Microsoft Excel 2016 یا جدیدتر (فقط برای باز کردن فایل خروجی)

حذف نصب
-------
از Settings → Apps → Installed apps → سازنده اکسل خالق: عرفان → Uninstall
یا از منوی شروع: Uninstall سازنده اکسل خالق: عرفان
